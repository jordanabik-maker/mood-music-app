## Project Description
A web application that recommends music based on a user’s mood using the Spotify API. Users can select a mood to instantly generate randomized songs with a responsive interface.

---

## Project Experience

### About the Project
I created this project to make it easier to find music based on how you're feeling. Instead of searching through songs manually, the app allows users to select a mood and instantly get music that matches it.

---

### How did your idea come about?
Some people can have a hard time finding songs that match their mood. I wanted to create a simple solution where users can select a mood and immediately get music recommendations.

---

### Challenges Faced
- Integrating the Spotify API and handling authentication  
- Debugging API errors such as 403 Forbidden and missing data  
- Making the user interface update without reloading the page  

---

### Problem-Solving Moment
One challenge I faced was figuring out how to upload my project to GitHub without exposing my private Spotify API credentials. I learned how to use a `.gitignore` file to keep sensitive information secure.

---

### What I Learned
- How to build a full-stack application using Flask  
- How to work with APIs and handle JSON data  
- How to improve user experience using JavaScript  
- How to manage and upload projects using GitHub  

---

### Experience
I enjoyed learning new technologies and building something from scratch. Although it was stressful at times, it was rewarding to see the final product working.
